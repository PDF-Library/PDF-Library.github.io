Users Guide On DevCloudAPI

Registering an account.

There are 3 subscription packages to select when registering a new account. With the Starter package being free of charge which contains a licensed version of the View-API.

The three package options will be displayed after registration:


  • Starter: Licensed ViewAPI for PDf viewing and rendering. Along with live analytics to track document viewing statistics.
  • Basic: View, Form and Annotation API. Provides advanced annotation tool, adding markups, comments, form filling and signing.
  • Professional: On top of Basic package, inclusion of Collaboration API with Real-time annotation. The simplest way to share the collaboration session with chats, comments and annotation exchanges.

Embed PDF viewer into a webpage





With DCA , it takes minutes to embed a PDF viewer for your website. The video involves github web page and our DevCloudAPI's View-API. The View API provides a simple read-only PDF Viewer, it is particularly useful when the purpose of the viewer is to mainly display documents without any interactivity required, such as for presentations, brochures, terms of use and other legal documents that do not require any type of interaction. The View API consists of:

  • PDF Viewing capabilities
  • Complete set of markups and annotation tools
  • The ability to Import/Export your annotation data to your local machine
The embedment is done with simple copying and pasting, without any programming required. For any future content upgrade, the process can be done easily by just replacing the file.

Adding Annotation capability to PDF-Viewer

Annotation API provides full PDF viewing and annotation capabilities.

  • Text Highlighting
  • Add Notes, Commenting
  • Free hand sketch-tool and other annotation tools
The user may repeat the same process as previous video for the View-API but selecting Annotation-API to embed a PDF-Viewer with annotation capabilities in their webpage.
Or they may follow the procedures in the video, by modifying the HTML code of the generated HTML tag created by DevCloudAPI on their web page.

PDF-File to PDF-Form

To change PDF-File into PDF-Form we recommend using ‘Foxit Phantom PDF’. This allows you to easily edit page/s of your PDFs by adding form fields on the file. Once modified, the newly made PDF-Form can be used with DevCloudAPI under the Form-API.

With added functions your new PDF form should contain fillable form fields.

Embedding PDF-Form into Webpage with notification on submissions.

Simple steps to embed a PDF-Form into your webpage.

Allowing users to fill and submit form with simple clicks and the administrator receiving email on user submissions. Email containing the content data submitted by user, as well as a CSV format of the content for easy data control.

Real-time Collaboration Session

With the use of DevCloudAPI's ‘Collaboration API’. You can turn your PDF viewer into an interactive document portal, where you can edit in real-time, communicate and even have conference meetings inside your document.



Sample of auto generated online collaboration HTML tag.

The video demonstrates the simple steps in which a user can embed their PDF file into their simple HTML web page. With the ability to host a zoom meeting within our DevCLoudAPI, through our Collaboration API.




The above video demonstrates the API in a side by side format to demonstrate the real-time annotation change function.

With the Collaboration API:

  • User receives all the features in View and Annotation API.
  • Newly joint users can input their own name.
  • Built in Live chat, improved efficiency by saving the user from having to open a third-party software.
  • Real-time annotation update.
