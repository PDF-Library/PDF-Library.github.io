PDF Library Easy steps involved using Developer Cloud API to embed PDFs in your web page.

  1. Visit the DCA Registration Page to make your account.
  2. Select a plan to obtain your licensed API key.
  3. Once directed to our SDK home page, select our Projects page.
  4. Select the "Deploy" option located next to the API Key of the default project.
  5. Choose the type of API you'd like to work with: View, Annotation, Form, Collaboration or Workflow.
  6. Select the iFrame option or embed with JavaScript if experianced with js.
  7. Paste in the necessary details i.e. PDF link, Page name etc. And customize the size and color scheme to your preference.
  8. Select "use Foxit DCA" to open link in a new tab and see how your PDF page turns out.
  9. Once satisfied, you may copy the link generated at the top of the iFrame page. And paste it into your own web HTML file.
  10. That’s it! You now have access to all of Foxit Developer Cloud API's powerful PDF library functions.

Foxit PDF SDK Library's features and different API functionalities:

Foxit's PDF-Library has a series of High-Quality features which allows our users to customize their PDFs with ease and accuracy.


Annotation API Offers a collection of commenting/annotation features:

  • Adding notes.
  • Series of different styled highlighting.
  • File attachement options including URLs, pictures and video/audio.
  • Free hand drawing, Stamps, Shapes etc.

Form API Offers view and Form submission capability:

  • Load up any PDF forms, which can be created with Foxit Phantom PDF.
  • Fill in the text field on form with ease. And select Submit Form when finished
  • An email containing the form content/data will be sent to the form owner. With inclusion of a CSV format for easy data analysis.

Collaboration API Offering you Real-Time online collabroative features:

  • Consisting all of our Form and Annotation API features.
  • With the inclusion of Edit Section which allows users to add any text, pictures or shapes to any PDF files. Aswell as the capability to edit them straight away or at a later stage.
  • With Collaboration API, any annotation change or editing change will be effective in real-time. If in a meeting, any changes made by "User A", can be seen immediately by everyone else.
  • Under Collaboration Section users can find a live group chat. Which contains video calling, or text messaging features. Allowing you to work closely with your team without having to open a third-party application.

Workflow API Introducing our brand new Magic Components::

  • A wide range of Magic Component features that you can embed in you PDFs
  • Reviewer Component which sends an email to your desired reviewer, they may review your forms and decide to approve or reject.
  • E_Sign Component offers you to include your E_signiture within the PDF file.
  • Add to Table and Add to Cart Component allows you to turn your PDF document into a data holding table. And can transform your PDF into a shopping Cart in seconds. And there is much more!
  • With our range of Magic Components, you can modify and create PDFs like never before. And save time by making use of all our features to complete your necessary tasks.
