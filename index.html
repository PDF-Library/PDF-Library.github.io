PDF-Library | PDF-SDK What is it

Application developers uses PDF-Library, the powerful, standards-compliant PDF technology to securely display, edit, annotate, format, organize, search PDF documents, fill PDF forms and to perform online collaboration in real-time.

PDF Library is an API for viewing, editing, collaborate and manipulating PDF files. PDF has become more and more recurrent and software companies aims to provide developers with complete PDF reader functionality, PDF content extraction, PDF Form functionalities and Cloud based Collaborarion features.

PDF SDK is a set of tools that help user to develop software that interacts with PDFs and PDF. The SDK - Software Development Kit contains type libraries, header files, sample code, documentation and simple utilities.
