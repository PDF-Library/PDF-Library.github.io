What is Foxit Developer Cloud API?
Foxit Developer Cloud API is a cloud-based platform for creating and embedding PDF viewers in a matter of minutes. It gives you access to Foxit PDF SDK for Web’s high-quality viewing/editing abilities without any need for development or server deployment. DevCloudAPI also allows you to customize the document viewer look and feel to add your own logo and colors, so you don’t even need a designer. Take a look at our short video here for more information.
What do the different APIs do?
DevCloudAPI is best used to add different PDF capabilities to your existing web site, internal web applications and/or your software products.
  • View API is for rendering PDFs in a web page.
  • Annotation API is used for commenting and markup on PDFs.
  • Form API is for PDF form submissions with email notifications also.
  • Collaboration API is for real-time PDF collaboration with multiple users. You can review documents with full annotation support also. Check out our video on APIs here.

To purchase DevCloudAPI, please contact our sales team here. If you select the Starter Plan, you will have licensed View API and also have trial versions of other APIs. If you want to use a licensed Annotation API, you must upgrade your subscription plan to either the Basic or Professional plan.

Yes, you can upgrade your subscription at any time. Simply navigate to the subscription options and select the package you'd like to upgrade to.
